# Template: Relatório de Validação Lean

**Template ID:** relatorio-validacao-lean-tmpl
**Versão:** 1.0
**Uso:** Entregáveis em Modo LEAN — decisão binária em 1-2 linhas
**Gerado por:** @ux-architect via *validar

---

**APROVA:**
```
APROVA — {razão comportamental em 1 linha}.
Teste dos 3 segundos: o cliente leva "{comportamento}" — comportamento-alvo confirmado.
```

**VETA:**
```
VETA — {elemento} cria {atrito} para {persona}.
Teste dos 3 segundos: o cliente leva "{X}" — falha no comportamento-alvo.
Alternativa: {ação específica e acionável}.
```

---

## Exemplo APROVA

```
APROVA — Anton "SEU CONTROLE" remove a barreira de autonomia do CEO desconfiante.
Teste dos 3 segundos: o cliente leva "controle" — comportamento-alvo confirmado.
```

## Exemplo VETA

```
VETA — fundo #F5A800 cria atrito de credibilidade para o CEO desconfiante.
Teste dos 3 segundos: o cliente leva "empresa barulhenta" — falha no comportamento-alvo.
Alternativa: #F5A800 como acento no CTA com fundo preto — remove atrito, preserva energia.
```

---

## Regras

- Máximo 3 linhas
- Sem vocabulário estético (bonito, clean, parece bom)
- Razão sempre comportamental: atrito, fluxo, conversão, persona
- Alternativa obrigatória em VETA
